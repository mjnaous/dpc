Discord Profile Checker

#ABOUT

This is a website to check discord users' profiles by providing their IDs.

This website uses JAPI.rest's API to get the searched user's info but I do NOT own JAPI.rest, I just use their API and you could too.

#FAQ

Q: Can I search a user by their username?

A: No, it is not possible to search a user by their username.


Q: Why does it shows a user offine even tho they're online?

A: Only users in JAPI.rest discord servers can be seen online, I can't do anything about that.
